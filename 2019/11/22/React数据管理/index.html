React数据管理

2019-11-22 React React 0 评论 字数统计: 106(字) 阅读时长: 1(分)

React数据管理

  • 依靠状态提升来和兄弟元素进行数据交互
  • 通过发布订阅模式做数据交互
  • Redux等数据管理工具

应用场景

  • 状态提升：组件层级扁平化，兄弟组件通信情况很少

  • 发布订阅模式：业务规模较小，层级较深的业务

  • Redux：业务复杂，组件层级较深，兄弟组件通信密切
